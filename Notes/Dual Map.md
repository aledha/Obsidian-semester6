``` ad-Definition
title: Definition.

_Let $V$ and $W$ be $K$-vector spaces and fix a linear map $T:V\to W$. The **dual map of $T$** is the linear map_
$$\begin{equation}
    T^\dual:W^\dual\to V^\dual\ \ \ \ \ \ \ \ \textrm{\it{mapping}}\ \ \ \ \ \ \ \ \gamma\mapsto\gamma\circ T.
\end{equation}$$
```
