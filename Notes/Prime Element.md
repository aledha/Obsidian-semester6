``` ad-Definition
title: Definition.

_Let $R$ be an integral domain. An element $p\in R$ is said to be **prime in $R$** if it is not a unit and if_
$$\begin{equation}
    \fa a,b\in R:p\divides\!\l(ab\r)\Rightarrow p\divides a\lor p\divides b.
\end{equation}$$
```
