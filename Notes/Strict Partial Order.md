``` ad-Definition
title: Definition.

_Let $X$ be a set. A binary relation $<$ on $X$ is a **strict partial order on $X$** if $<$ is transitive and irreflexive on $X$._

```

**Remark.** Write $>\,\coloneqq\,<^{-1}$.<span style="float:right;">$\blacklozenge$</span>
