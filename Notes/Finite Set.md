``` ad-Definition
title: Definition.

_A set $X$ is said to be **finite** if $X\eqnum n$ for some $n\in\omega$._

```

**Remark.** Here, $n=\l\{0,\dots,n-1\r\}$ is interpreted set-theoretically.<span style="float:right;">$\blacklozenge$</span>
