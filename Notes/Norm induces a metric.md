``` ad-Proposition
title: Proposition.

_Let $K$ denote either $\R$ or $\C$ and let $V$ be a normed vector space over $K$. Then the function_
$$\begin{equation}
    d:V\times V\to\R\ \ \ \ \ \ \ \ \textrm{\it{mapping}}\ \ \ \ \ \ \ \ \tpl{x,y}\mapsto\|x-y\|
\end{equation}$$
_is a metric on $V$._

```
