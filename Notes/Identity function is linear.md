``` ad-Proposition
title: Proposition.

_Let $V$ be a $K$-vector space. Then $\id_V$ is a linear map on $V$._

```

_Proof_. Take $v,w\in V$ and $\alpha\in K$, and observe that
$$\begin{equation}
    \id_V\l(\alpha v+w\r)=\alpha v+w=\alpha\id_V\l(v\r)+\id_V\l(w\r).\qedin
\end{equation}$$
