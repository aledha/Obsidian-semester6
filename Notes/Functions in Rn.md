``` ad-Definition
title: Definition.

_Let $\R$ be the set of real numbers and fix $m,n\in\N^+$. A function $f$ is said to be a **multivariate function of a vector variable** if $f:S\to\R^m$ for some $S\subseteq\R^n$._
```
