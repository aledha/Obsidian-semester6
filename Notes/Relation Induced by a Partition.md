``` ad-Definition
title: Definition.

_Let $X$ be a set and let $\Pi$ be a partition on $X$. The **relation induced by $\Pi$** is the relation_ $R_\Pi$ _defined by_
$$\begin{equation}
    xR_\Pi y\ \ \ \ \Leftrightarrow\ \ \ \ \ex P\in\Pi:x\in P\land y\in P
\end{equation}$$
_for all $x,y\in X$._

```
